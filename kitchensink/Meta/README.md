# Meta

Adds commands for manipulating a held item.

| Subcommand | Function                        | Usage                 |
|------------|---------------------------------|-----------------------|
| `author`   | Set a book's author             | `/meta author <name>` |
| `title`    | Set a book's title              | `/meta title <title>` |
| `name`     | Set an item's display name      | `/meta name <name>`   |
| `add`      | Add a line of lore              | TODO                  |
| `set`      | Change an existing line of lore |                       |
| `insert`   | Insert a line of lore           |                       |
| `owner`    | Change a player head's owner    |                       |
| `texture`  | Change a player head's texture  |                       |
| `delete`   | Remove a meta entry             |                       |
| `clear`    | Reset the metadata to defaults  |                       |
